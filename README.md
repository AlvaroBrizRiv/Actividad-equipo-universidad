# 📰 El Faro Chile - Portal de Noticias

¡Hola! 👋 Bienvenidos al repositorio de **"El Faro Chile"**, 
un proyecto universitario colaborativo enfocado en el aprendizaje y aplicación de desarrollo web estructurado.

----------------------------------------------------------------------------------------------------------

## 🎯 Objetivo del Proyecto
Esta actividad fue desarrollada como parte de nuestra formación práctica en **Programación y Análisis de Sistemas**. 
El propósito principal es aplicar los conceptos fundamentales de estructuración mediante HTML5, creando un periódico 
digital interactivo y bien organizado. 

----------------------------------------------------------------------------------------------------------

## 🚀 Características Principales
* **Estructura Semántica:** Construcción limpia utilizando jerarquía de títulos (`<h1>`, `<h2>`) y secciones claras.
* **Navegación Interna (Menú):** Enlaces ancla (`<nav>`) que permiten al usuario saltar rápidamente a las categorías
  de **Deportes ⚽** y **Negocios 💼**.
* **Integración Multimedia:** Inserción de imágenes para acompañar cada titular y dar vida a las noticias.
* **Redirección de Fuentes:** Enlaces externos hipervinculados para que los lectores puedan profundizar en los
  artículos originales.

-----------------------------------------------------------------------------------------------------------
